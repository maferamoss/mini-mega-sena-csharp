# 🎲 Mini Mega Sena em C#

Projeto acadêmico desenvolvido em C# como parte dos estudos em Análise e Desenvolvimento de Sistemas.

O programa simula um jogo da Mega-Sena em modo console, permitindo ao usuário escolher números e comparar com o sorteio gerado aleatoriamente pelo sistema.

## 🧠 Sobre o projeto
A aplicação trabalha lógica de programação e validações, simulando regras básicas de um jogo de apostas.

## 🚀 Funcionalidades
- Escolha de números pelo usuário
- Geração automática de números sorteados
- Comparação entre aposta e sorteio
- Exibição do resultado final

## 🛠 Tecnologias utilizadas
- C#
- .NET
- Aplicação Console

## ▶️ Como executar
1. Abra o projeto no Visual Studio
2. Execute o programa com `Ctrl + F5`
3. Siga as instruções exibidas no console

## 📚 Contexto acadêmico
Projeto desenvolvido para fins educacionais, com foco em lógica de programação e prática em C#.
